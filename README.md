# The Zipped Finite Element Method: High-order Shape Functions for Polygons

This C++ code generates examples published in "The Zipped Finite Element Method: High-order Shape Functions for Polygons" by S. Berrone, L. Neva, M. Pintore, G. Teora, and F. Vicini.
